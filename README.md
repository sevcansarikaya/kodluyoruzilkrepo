### Küçük Bir İpucu
Daha önceki mesajında aldığın "divergent branches" hatasını çözmek için terminale şu komutu yazıp **Enter**'a basman yeterli olacaktır:
`git config pull.rebase false`

Ardından `git pull origin main` yaparak GitHub'daki mevcut dosyaları bilgisayarına çekebilir ve bu hazırladığımız `README.md` içeriğini ekleyip tekrar push edebilirsin.

Dosyayı oluşturup kaydettikten sonra VS Code'da nasıl göründüğüne bakmak ister misin? (Sağ üstteki **Preview** butonuna basarak görebilirsin).
